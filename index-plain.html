LED Wiring & Integration

Power, control, and illumination setup

Components

  • 🔌 5V LED Strip (WS2812B addressable)
  • 📌 Arduino Nano microcontroller
  • 🔋 5V/2A USB Power Supply
  • 🎛️ PWM Control Module
  • 🔗 Soldered connections (24 AWG wire)

Wiring Diagram

Implementation Notes

The LED strip is controlled via Arduino Nano running FastLED library, allowing for dynamic color animations and effects. The addressable LEDs (WS2812B) are wired in series and powered directly from a 5V USB supply for reliable operation. Resistor (330Ω) placed between Arduino data pin and LED data input to ensure signal integrity.

Project Reflection

Insights, challenges, and lessons learned

✓

What Worked Well

  • Modular design allowed for easy assembly and disassembly
  • LED integration was seamless with proper planning
  • Print quality exceeded expectations with optimal settings
  • Cable management system kept everything organized
⚠

Challenges Faced

  • Initial LED power supply was insufficient for full brightness
  • Support removal required careful post-processing
  • Tight tolerances needed multiple calibration prints
  • Wiring complexity demanded patience and precision
→

Future Improvements

  • Add Bluetooth control for wireless color selection
  • Implement temperature sensing for power management
  • Redesign for multiple size options
  • Create a custom control app for iOS/Android

This project demonstrated the convergence of design thinking, engineering precision, and creative problem-solving. By combining CAD modeling, 3D printing, and embedded electronics, we created a piece that's both functional and aesthetically striking. The Orion Lightbox stands as a testament to what's possible when we merge multiple technologies into a cohesive, elegant solution.
